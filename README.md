# OnlyAdmin
A Admin framework let you time free!

## Dependencies

- Django
- AdminLTE

## How to Start

```
virtualenv .env -p python3
source .env/bin/activate
pip install -r requirements.txt

cd OnlyAdmin
./manage.py runserver
```
